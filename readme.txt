Attach subscription options to existing WooCommerce products.

== Description ==

WooCommerce Subscribe to All the Things is a mini-extension for [WooCommerce Subscriptions](http://www.woothemes.com/products/woocommerce-subscriptions/) that provides experimental subscriptions support for non-subscription product types. It allows you to attach subscriptions to Simple products, Variable products, [Composite Products](http://www.woothemes.com/products/composite-products/), [Product Bundles](http://www.woothemes.com/products/product-bundles/) and [Mix and Match Products](http://www.woothemes.com/products/woocommerce-mix-and-match-products/).

**Requires WooCommerce Subscriptions v2.0+**.

== Installation ==

1. Upload the plugin to your **/wp-content/plugins/** directory
2. Activate the plugin through the **Plugins** menu in WordPress
